# 旅行者(岩)

## 圣遗物套装  

- 宗室4：通用毕业套装，偏功能性。  

- 岩2+宗室2/攻击力2：通用毕业套装，偏输出。  

- 磐岩4  

- 流放者4：辅助部分缺充能主c(如诺艾尔)专用。  

## 圣遗物对应属性  

- 暴击/暴伤头，攻击沙，岩伤杯。  

  > 提示：辅助圣遗物无需参考对应属性及有效词条。  

## 圣遗物有效词条  

- 攻击力、暴击率、暴击伤害。
