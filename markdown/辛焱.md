# 辛焱

## 圣遗物套装  

- 千岩4：盾辅专用套装。  

- 流星4：六命主c专用套装。  

- 骑士4：六命主c专用套装。  

- 苍白2+骑士2/攻击力2：大招流副c专用。  

- 华馆4：纯盾辅专用套装。  

## 圣遗物对应属性  

- 防御头，防御沙，防御杯：辅助。  

- 暴击/暴伤头，攻击沙，物伤杯：脱手副c及主c。  

- 暴击/暴伤头，防御沙，物伤杯：主打重击的六命主c或搭配赤角石溃杵使用。  

## 圣遗物有效词条  

- 辅助：防御力、充能。  

- 主c：攻击力、暴击率、暴击伤害。  

  > 注：重击流主c或搭配赤角石溃杵时(六命)，防御力也为有效词条。
