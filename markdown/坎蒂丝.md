
# 坎蒂丝

- 千岩2+绝缘2：通用毕业套装。
- 绝缘4：副c专用毕业套装。
- 宗室4：功能性较强。

## 圣遗物对应属性：
- 生命头，充能沙，生命杯：通用。
- 生命头，生命沙，生命杯：极限拐，建议搭配充能武器使用。
- 暴击/暴伤头，生命沙，水伤杯：副c专用。

## 圣遗物有效词条：
- 充能、生命值。
